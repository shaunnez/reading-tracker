## Schema Changes

```typescript
// src/lib/db/schema.ts

// phonicsSkills table:
stage: text("stage").notNull()         // replaces: phase: integer
                                        // values: "1","2","3","4","4+","5","6","7.1","7.2","7.3","7.4"
heartWords: text("heart_words")        // new: comma-separated heart words for this stage

// resources table:
stageMin: text("stage_min").default("1")   // replaces: phonicsLevelMin: integer
stageMax: text("stage_max").default("7.4") // replaces: phonicsLevelMax: integer
```

## Execution Waves

### Wave 1 — Write content in parallel (no DB needed)
All 4 workers run simultaneously.

**Worker A — Schema**
- `src/lib/db/schema.ts`: phase → stage, add heartWords, update resources level fields
- Generate Drizzle migration SQL

**Worker B — Seed Data**
- Rewrite `src/lib/db/seed.ts`:
  - 63 phonicsSkills with stage, heartWords, warmup, introduction, wordList, dictationWords, tipsForParents
  - LLLL-methodology word lists per skill (decodable CVC→VCe→digraph progressions)
  - 11 Pip and Tim book resources + update existing resources with stageMin/stageMax

**Worker C — PHONEME_VISUAL**
- Rewrite `PHONEME_VISUAL` in `src/app/phonics/[id]/page.tsx`
- 63 entries, one per skill
- Emoji, sound, keyword, heroBg, heroText, highlight per skill
- Color palette by stage (stage 1=warm amber, stage 2=sky blue, stage 3=emerald, stage 4=violet, 4+=rose, stage 5=orange, stage 6=teal, stage 7=purple gradient)

**Worker D — phonics-map.tsx**
- Replace phase grouping with stage grouping
- STAGE_INFO: 11 stages with label, color, description
- Handle "4+" stage slug in keys/filters
- Update progress bars and mastery counts

### Wave 3 — Update consumers in parallel (after seed confirmed)

**Worker E — actions.ts**
- `getPhonicsSkills()`: return stage field instead of phase
- `applyPlacementResult()`: map result to stage text value
- `getDashboardStats()`: currentStage instead of currentPhase
- `getResources()`: query by stageMin/stageMax text comparison

**Worker F — Placement test**
- `src/app/phonics/placement/placement-test.tsx`
- STAGE_LABELS replacing PHASE_LABELS
- Adaptive algorithm: stage-aware skip logic
- Result maps to stage text ("1" through "7.4")

**Worker G — Verbiage sweep**
- `src/app/page.tsx`: "Phase X" → "Stage X", currentPhase → currentStage
- `src/components/nav-bar.tsx`: any phase references
- `src/app/progress/progress-charts.tsx`: phase axis → stage axis
- `src/app/books/page.tsx` + `book-form.tsx`: phonicsLevel → stage selector
- `src/app/sessions/new/session-form.tsx`: phase display → stage display
- Global text: "Phase" → "Stage", "months 1-3" → stage names
